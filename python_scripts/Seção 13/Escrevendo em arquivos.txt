"""
Escrevendo em arquivos

Observação: Ao abrir um arquivo para leitura, não podemos realizar a escrita nele, apenas ler.
Da mesma forma, se abrirmos um arquivo para escrita, não podemos lê-lo, somente escrever nele.

Observação: Ao abrir um arquivo para escrita, o arquivo é criado no sistema operacional.

Para escrevermos dados em um arquivo, após abrir um arquivo utilizamos a função write().
Essa função recebe uma string como parâmetro, caso contrário teremos um TypeError.

Abrindo um arquivo para escrita com o modo 'w', se o arquivo não existir será criado, caso
ele já exista, o anterior será apagado e um novo será criado. Dessa forma, todo o conteúdo
do arquivo anterior é perdido.



# Exemplo de escrita - modo 'w' - write (escrita/escrever)

with open('novo.txt', 'w', encoding='UTF-8') as arquivo:
    arquivo.write("Novos dados.\n")
    arquivo.write("Outros podemos colocar quantas linhas quisermos.\n")
    arquivo.write("Mais esta é a última linha.\n")

with open('novo.txt', encoding='UTF-8') as arquivo:
    print(arquivo.read())
print(arquivo.closed)

# Forma tradicional de escrita em arquivo (Não Pythônica)
arquivo = open('mais.txt', 'w', encoding='UTF-8');
arquivo.write("Um texto qualquer. \n")
arquivo.write("Mais um texto. \n")
arquivo.close()
--------
with open("luccas.txt", "w") as arquivo:
    arquivo.write("Geek" * 1000)
--------
"""

with open('frutas.txt', 'w', encoding="UTF-8") as arquivo:
    while True:
        fruta = input("Informe uma fruta ou digite sair:")
        if fruta != "sair":
            arquivo.write(fruta + '\n')
            # arquivo.write("\n")
