"""
Leitura de arquivos

Para ler o conteúdo de um arquivo em Python utilizamos a função integrada open() que literalmente significa abrir.
open() -> Na forma mais simples de utilização nós passamos apenas um parâmetro de entrada, que neste caso é caminho
para o arquivo ser lido. Essa função retorna um _io.TextIOWrapper e é com ele que trabalhamos então.

https://docs.python.org/3/library/functions.html#open

Observação: Por padrão, a função open() abre o arquivo para leitura. Esse arquivo deve existir, ou então teremos
o erro FileNotFoundError.
"""
# Exemplo

arquivo = open('texto.txt', encoding='UTF-8')
# print(arquivo)  # <_io.TextIOWrapper name='texto.txt' mode='r' encoding='cp1252'>
# print(type(arquivo))  # <class '_io.TextIOWrapper'>

# Para ler o conteúdo de um arquivo, após sua abertura, devemos utiliza a função read();
print(arquivo.read())

# Observação: O Python utiliza um recurso para trabalhar com arquivos cahamado cursor. Ele funciona com o cursor
# quando estamos escrevendo.
